# Tanner Turkey Devourer

A premium fictional parody ecommerce experience.

**One Turkey. Zero Leftovers.**

## Website

This repository contains a responsive static website designed for iPhone, tablet, and desktop. It includes a fictional merchandise collection, product filters, quick-view modals, wishlist buttons, an interactive parody cart, a no-payment checkout ending, reviews, awards, news cards, gallery, timeline, FAQ, custom 404 page, and Turkey Mode easter egg.

No real merchandise is sold. No payments or customer information are collected.

## GitHub Pages

Publish from GitHub Pages using **GitHub Actions**. The deployment workflow is included at `.github/workflows/pages.yml`.
